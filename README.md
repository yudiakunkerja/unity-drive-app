# ☁️ Unity Drive - Multi-Cloud Aggregator

Unity Drive adalah dashboard penyimpanan terpadu untuk mengelola file lintas cloud dengan satu akun Google.

## 🚀 Fitur
- **SSO Login:** Masuk cepat dengan Google.
- **Smart Expansion:** Tambah 1 TB dari TeraBox secara instan.
- **Auto-Routing:** File besar otomatis diarahkan ke storage yang paling luas.

## 📁 Struktur File
- `index.html`: Gerbang login.
- `dashboard.html`: Antarmuka manajemen file.
- `app.js`: Logika deteksi kapasitas dan upload.

---
*Proyek ini dibangun untuk optimasi penyimpanan cloud gratis.*
