# ☁️ Unity Drive - Multi-Cloud Aggregator

Unity Drive adalah platform inovatif yang memungkinkan pengguna menggabungkan berbagai layanan penyimpanan cloud (seperti Google Drive dan TeraBox) ke dalam satu dashboard terpadu hanya menggunakan **satu akun Google**.

## 🚀 Fitur Unggulan
- **Single Sign-On (SSO):** Login satu kali dengan Google untuk akses ke semua layanan.
- **Auto-Expansion:** Dapatkan instan 1 TB tambahan dengan menghubungkan TeraBox via Google.
- **Smart Distribution:** Sistem otomatis membagi file besar ke storage yang lebih luas.
- **Unified Explorer:** Lihat semua file dari berbagai cloud dalam satu daftar.

## 🛠️ Teknologi yang Digunakan
- **Frontend:** HTML5, Tailwind CSS
- **Icons:** Font Awesome
- **Authentication:** Google OAuth 2.0 (Simulasi)

## 📂 Struktur Proyek
- `index.html` - Halaman Login Utama.
- `dashboard.html` - Dashboard pengelolaan file dan cloud.
- `README.md` - Dokumentasi proyek.
